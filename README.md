# POO-em-JAVA
Realizei vários desafios do curso de POO em Java (Alura) e fiz dois projetos usando os conhecimentos adquiridos no curso

Os arquivos AudioVerse e ScreenMatch, foram os projetos realizados durante o curso de POO em Java.
O Screen Match é similar á um serviço de Streaming, utilizando de polimorfismo, interfaces, métodos...
O Screen Match separa a classificação (avaliação do filme ou série), a duração em minutos, quantidade de episódios, nome, ano de lançamento, etc.

O AudioVerse foi um projeto menor, para aprender sobre polimorfismo e métodos, que consistem em separar e classificar músicas e podcasts, de acordo com quantidade de curtidas ou reproduções.

Os outros arquivos são desafios mais simples, eles são bem auto explicativos por si só, mas basicamente utilizei o POO para realizar tarefas simples.

Gostei muito de fazer esse curso, pois aprendi a programar em JAVA para simplificar métodos, e muita das vezes deixar meu código menos bagunçado, e mais eficiente.
